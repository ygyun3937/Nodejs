# Nodejs

1. 설치
   - https://nodejs.org/en
   - "LTS" 최신 버전 다운로드 설치
    
2. 설치 확인
   - console 활용
     - 명령어
        - 설치 버전 확인

        - node 환경 진입

            - 예제 (1+1 콘솔에 보이기)
               -> console.log(1+1)

          - 단축키 : Ctrl+C 2번
